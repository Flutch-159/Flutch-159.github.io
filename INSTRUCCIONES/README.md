## Requisitos
- PHP >= 8.1.6 https://windows.php.net/download#php-8.1
(NOTA: descargar VS16 x64 Thread Safe (2024-Sep-25 08:54:58))

- Composer https://getcomposer.org/download/

- XAMPP (o cualquier servidor local con Apache y MySQL)

- MySQL
